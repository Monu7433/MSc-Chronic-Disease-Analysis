# MSc Chronic Disease Analysis (SAS & SPSS)

## Project Overview
This project is part of my M.Sc final year work focusing on chronic disease analysis using healthcare data. The study includes descriptive statistics, visualization, correspondence analysis, chi-square tests, and binary logistic regression modeling to understand disease prevalence, associations, and risk factors.

---

## Objective
- Analyze prevalence of chronic diseases
- Study association between different diseases
- Perform correspondence analysis to understand relationship patterns between categorical variables
- Identify risk factors using binary logistic regression
- Apply statistical modeling techniques for healthcare data analysis

---

## Methods Used
- Frequency analysis
- Data visualization (bar plots)
- Correspondence analysis
- Chi-square test
- Cramer's V
- Binary logistic regression
- Stepwise model selection

---

- SAS
- SPSS
- Microsoft Excel
